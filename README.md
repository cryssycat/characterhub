# StarSundae Character Hub

Upload these files to GitHub or Cloudflare Pages.

## Edit characters
- Homepage cards are in `index.html`.
- Character pages are in `characters/`.
- Put images in `images/character-name/`.

Suggested image names:
- `icon.png`
- `main.png`
- `gallery-1.png`
- `gallery-2.png`
- `gallery-3.png`
